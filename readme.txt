Given a discrete data set, the calculator calculates the mean, variance, and standard deviation of a sample or a population and shows all the intermediate steps of calculations.

[https://www.calculator.io/standard-deviation/](https://www.calculator.io/standard-deviation/)

== Usage ==

Add the Standard Deviation shortcode to your page, post or sidebar:

`[ci_standard_deviation]`

Add the following code to your template where you would like the Standard Deviation to appear:

`<?php display_ci_standard_deviation(); ?>`

== Installation ==

1. Upload the Standard Deviation /ci_standard_deviation/ folder to the /wp-content/plugins/ directory.
2. Activate the Standard Deviation plugin through the "Plugins" menu in WordPress.
